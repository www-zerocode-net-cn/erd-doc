## 新版本用一键脚本安装完之后，点击登录没反应
- 首先，在浏览器上查找前端调用信息：以谷歌浏览为例：按F12，打开调试窗口，打开`网络`这一tab项，再次点击登录，查看返回信息
<br/>
- 然后，登录服务器，也就是执行一键部署脚本的服务器，执行以下命令，查看服务信息
```shell
docker-compose logs -f
```
- 最后，根据前后端信息，大致能定位到具体问题
